# LLM Response Evaluation with Simple Metrics

This example demonstrates how to move from subjective LLM evaluations to objective, metric-driven processes. It simulates an LLM's response and then applies simple, quantifiable metrics like keyword presence and response length to assess its performance against predefined criteria. This highlights the importance of automated evaluation for production-level LLM applications.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
